In-app messaging guard — escalation demo

Type a message and press Send. Clean messages go through. A message containing a flagged term triggers the pop-up. Each flagged attempt in this thread escalates: 1st → educate, 2nd → warn, 3rd+ → blocked. This is the prototype (client-side) layer.

Try: can you venmo me · text me at 555-123-4567 · let's cancel and pay in cash · my email is a@b.com · sounds good, see you at 3 (clean, passes)
